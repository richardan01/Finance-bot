# Finance Bot 💰

AI finance assistant prototype for structured analysis, reasoning, charting, and human review.

## What this is

Finance Bot explores how AI can support finance and operations users by turning unstructured questions into structured, reviewable answers.

## What it demonstrates

- AI assistant UX
- Structured reasoning
- Finance workflow support
- Human-in-the-loop review
- Charting and analysis
- Eval and failure-mode thinking

## Product thinking

Problem → Data → Assistant response → Human review → Decision support

## Status

Prototype / learning project.
